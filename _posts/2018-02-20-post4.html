How much ram do you really need? 8gb? 16gb? 64gb? 

Short answer

As much as you can afford, because more is always better, and the additional power draw is negligable in this case.

Long Answer-

If you're shopping for a laptop, you are likely to see ads for laptops that have usually 8gb or 16b configurations.  If price is no object, then it usually makes sense to buy a machine with as much ram as possible to 'future-proof' it as much as possible.  However, quite often, the extra ram may not be useful if you don't actually use it for anything.  For web browsing and watching cat videos on youtube 8gb, or even 4gb on a windows machine will be more than sufficient.  Most smartphones that are released in 2018 actually tend to have less than 4gb of ram as well.  One of the differences between phones and desktops is that phones are more aggressive at suspending, or terminating unused applications in the foreground.  Usually the phones that tend to have more ram have more advanced capabilities to run applications in true multitasking containers.

Personally, I buy the largest sticks of ram possible that are accepted by my motherboard.  I currently have 2x16gb of ram in my desktop.  However, I have never come close to utilizing even half of it.   But of course, I have a habit of closing unused applications.  At the time of writing, I am running a virtual box instance, and a chrome browser on a Windows 10 host machine.  Task manager reports that chrome is using 2.3gb of ram, while the virtual box manager is using ~0.1gb.  It seems that the usage by  ubuntu virtual machine does not appear in windows Task Manager.  This should be something that could need to be fixed or it might be a 'feature' of virtualbox.
